# Toncoin P2P Monitor

Наблюдает за лучшими SELL‑объявлениями Toncoin на официальном P2P API и предупреждает, если цена опустилась ниже заданного порога.

## Что делает
- читает ключ `P2P_API_KEY`, валюты и тайминги из окружения;
- опрашивает `https://p2p.walletbot.me/p2p/integration-api/v1/item/online`;
- отправляет уведомления в Telegram, если курс упал на `P2P_DROP_PCT` или ниже;
- отдельно отправляет новый лот, как только он появляется в выборке подходящих объявлений;
- после предупреждения пересчитывает эталон и ждёт следующего провала;
- фиксирует ошибки API и продолжает работу.

По умолчанию бот каждые `90` секунд поддерживает одно сообщение с `3` текущими доступными лотами в диапазоне `500–5000 RUB` и редактирует его при изменениях.
Под сообщением в Telegram появляются только кнопки `Лот 1/2/3` для прямого перехода в конкретные лоты.
Когда ранее отправленный новый лот пропадает из выборки, бот редактирует то сообщение и помечает его как `Продано`.
Основной отчёт по текущим лотам тоже редактируется: пропавшие лоты уходят из списка текущих и попадают в блок `Продано`.

## Как запускать

1. Создайте виртуальное окружение и установите зависимости:

   ```powershell
   python -m venv .venv
   .\.venv\Scripts\activate
   pip install -r requirements.txt
   ```

2. Определите обязательный ключ:

   ```powershell
   set P2P_API_KEY=wZXVPe5LAgW46G8VLPIs6Fpjsbly3yK25wWU
   ```

   (или добавьте переменную в конфигурацию PyCharm).

3. При необходимости задайте дополнительно:

   ```powershell
   set P2P_FIAT=RUB
   set P2P_CRYPTO=TON
   set P2P_INTERVAL=90
set P2P_DROP_PCT=0.5
set P2P_DROP_AMOUNT=500
   set P2P_LOT_REPORT_LIMIT=3
   set P2P_PAGE_SIZE=20
   set TELEGRAM_BOT_TOKEN=none сюда бота
   set TELEGRAM_CHAT_ID=none сюда чат/группу
   set P2P_MIN_LOT_LIMIT=500
   set P2P_MAX_LOT_LIMIT=5000
   ```

   - `P2P_DROP_AMOUNT` (по умолчанию 500) позволяет отслеживать абсолютное падение в рублях, чтобы приходило сообщение при падении цены минимум на эту сумму.  
   - `P2P_LOT_REPORT_LIMIT` задаёт сколько лотов попадёт в сообщение/лог; по умолчанию отправляются `3` лота.  
   - `TELEGRAM_CHAT_ID` может быть отрицательным (групповой чат `-100...`), не забудьте добавить бот в группу и дать ему права на отправку сообщений.  
   - `P2P_MIN_LOT_LIMIT` задаёт нижнюю границу диапазона лотов; по умолчанию это `500 RUB`.  
   - `P2P_MAX_LOT_LIMIT` задаёт верхнюю границу диапазона лотов; по умолчанию это `5000 RUB`.  
   - `P2P_REPORT_EVERY_CYCLE=1` включает автоматическую рассылку "Текущий курс" каждые `P2P_INTERVAL` секунд; по умолчанию это `90` секунд.  

4. Запустите монитор:

   ```powershell
   & ".\.venv\Scripts\python.exe" -m ton_monitor
   ```

5. Чтобы сразу отправить текущий список лотов (не дожидаясь падения) в группу/бот, выполните:

    ```powershell
    python -m ton_monitor --send-now
    ```

6. Также можно просто написать `\/crypto` боту или в группу — в ответ появится отчёт по текущим лотам в диапазоне `500–5000 RUB` и со ссылками на объявления.

5. Если вы предпочитаете PyCharm, добавьте те же переменные в **Run → Edit Configurations**, выберите interpreter `.venv\Scripts\python.exe` и в поле **Module name** укажите `ton_monitor`.
и введите все переменные выше перечисленные 

## Возможные улучшения

- добавить веб-интерфейс или API для просмотра последней цены;
- сохранять состояние (последний эталон) между перезапусками;
- подключить другие каналы уведомлений (email, webhook).
